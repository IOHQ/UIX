UIX Framework0.1

Introduction

Yes, it's another frontend framework.

I know what you're thinking, We have plenty of these already, bro.. Here me out though, because this framework focuses on some niche points that others don't - standards!

This framework is designed in a way that rather you're making a portfolio, company website, product page or full web application, themes can be switched out so that no matter the purpose, changing things up can be as easy as changing a link tag!

Grammar

$.camel

return $.camel("My Name Is Paul!");

$.capitalize

Capitalize the first letter of a string.

$.getAlphaNumeric

Return only alpha-numeric characters.

Tools

$.copy

Create a copy of an object.

var MyObject = { "a": 1 };
var NewObject = $.copy(MyObject);
NewObject.a++;

return JSON.stringify($log(MyObject), false ,2);

$.logging

Enable a function-specific logging function.

var $log = $.logging('My Category');
$log("Hello world!");

When passing multiple arguments, the console will create a group of the arguments prefixed by their type.

// Post a simple log.
function MyFirstFunction(){
    var $log = $.logging('MyFirstFunction');
    $log('1+2 is ' + (1+2));
}

// Log something in groups.
function mySecondFunction(){
    var $log = $.logging('mySecondFunction');
    $log('1+1 is:', {answer:1+1}, 'but it is not: ' + (2+2));
}

MyFirstFunction();
mySecondFunction();

$.type

Get specific details on some data.

$.type(5.2);
//  result -> {type: "float", native: "number", isNumber: true, isFloat: true, isOdd: true}
